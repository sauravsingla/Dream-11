# Support

## Getting help

Before opening an issue:

1. Read the installation and usage instructions in `README.md`.
2. Review the expected CSV schema in `data/README.md`.
3. Check `docs/data_provenance.md` for dataset limitations.
4. Run the included synthetic example with PuLP/CBC.
5. Search existing issues for a similar question or error.

When requesting help, include:

- operating system and Python version;
- package and solver versions;
- the exact command used;
- the complete error message;
- a minimal reproducible input that contains no private or licensed data.

## Solver-specific support

PuLP with CBC is the default open-source backend. Gurobi requires a separate valid licence. Questions about Gurobi installation or licence activation should be directed to Gurobi's official support channels.

## Research and reproduction questions

Exact reproduction of the article requires the original player scores and prices, which are not currently distributed in this repository. The included dataset is synthetic and demonstrates the workflow only.

Questions and reproducible bug reports may be submitted through GitHub Issues. Please do not submit credentials, private datasets, commercial solver licence files or personally identifiable information.
